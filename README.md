# 🌐 Web Portfolio

A modern, responsive, and visually stunning personal portfolio website built using core web technologies and enhanced with dynamic interactive 3D background animations.

---

## 🚀 Features

* **Dynamic 3D Animations:** Powered by **Vanta.js** and **Three.js** for an immersive, modern user experience.
* **Fully Responsive Layout:** Built from scratch using **HTML5** and **CSS3** (Flexbox/Grid) to ensure seamless viewing across mobile, tablet, and desktop screens.
* **Vector Iconography:** Integrated with **Font Awesome** for clean, scalable social and technical stack icons.
* **Interactive Location Map:** Embeds an interactive map component showcasing geographic location/contact context.
* **Smooth Navigation:** Clean animations and transitions for page UI interactions.

---

## 🛠️ Built With

* **HTML5** - Structured semantic markup
* **CSS3** - Custom styling, layouts, and standard keyframe animations
* **Vanta.js & Three.js** - Advanced JavaScript libraries for the animated background effects
* **Font Awesome** - Scalable vector icon library
